* dayjs( ) - 날짜 함수 
 -> 현재 날짜/시간 : dayjs( )
 -> 날짜 파싱 : dayjs('2026-02-25') or dayjs(dateString)
 -> 포맷팅 : .format( 'YYYY-MM-DD HH:mm:ss' )
 -> 더하기/빼기 : .add(7, 'day'), .subtract(1, 'month')
 -> 차이 계산 : .diff(compareDAte, 'unit') => 일, 시간, 분 단위
 -> 년도 :  'YYYY' : 2026 / 'YY' : 26 
 -> 월 : MMMM : 영어로 풀 네임(January) / MMM : 영어 약어(Jan) / 'MM' : 02 / 'M' : 2 
 -> 일 : 'DD' : 01 / D : 1
 -> 시 : HH : 00 - 23 시(24시간) / hh : 01 - 12 시(12시간)
 -> 분 : mm : 00 - 59 분 / m : 0 - 59분
 -> 초 : ss : 00 - 59초 / s : 0 - 9초
 -> 오전/오후 : A : AM / PM , a : am / pm

* 내 장바구니에 상품이 있는지 조회하는 쿼리에서 int로 리턴하는게 위험한 이유
 -> 조회가 안될 때 데이터가 없다는 뜻으로 null이 나오는데 int에 null이 들어갈 수 없음!
 -> int 뿐만 아니라 자바 기본 자료형엔 null이 안들어감!
 --> 그래서 참조자료형으로 써주는게 안전함.
 --> Wrapper class : Integer(int와 똑같은데 null이 들어감)
