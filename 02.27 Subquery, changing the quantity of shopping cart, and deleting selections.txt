<체크박스 선택삭제>
 - 체크박스 선택 : [1, 2, 3] 이렇게 배열 형태로 넘겨야 함 -> 체크박스 자체가 배열이니까

*스프링*
 1. 체크박스 선택 삭제 쿼리 작성 -> delete
 - 체크박스 선택삭제 쿼리에서는 어떤 장바구니를 삭제할 것인지 조건으로 지정
   -> 체크박스를 선택할 때 하나만 선택하는 것이 아니라 여러 개를 선택할 수 있기에 변수가 반복적으로 여러 번 저장될 수 있도록 <foreach>문 활용
  -> <foreach>문에는 여러 속성이 있음!
      => 	collection : 변수명, 
	item : 내가 반복으로 필요한 데이터, 
	separator : for문 결과 나오고 반복(마지막 반복은 실행 X), 
	open : 반복 시작후 최초 한번실행, 
	close : 반복 끝나고 최초 한 번 실행

 2. 속성 넣어주고 반복해서 저장할 변수 넣어주기

 ex) <delete id="deleteCarts">
        DELETE FROM SHOP_CART
        WHERE CART_NUM IN
        <foreach collection="cartNumList" item="cartNum" separator="," open="(" close=")"> -----> 속성들
            #{cartNum} -----------------> 반복문을 활용해 데이터를 저장할 변수
        </foreach>
    </delete>

 3. Mapper 작성 
  - 매개변수자리에 쿼리의 빈 값을 메꿔주는 변수를 넣어야 하는데 반복문이라 매개변수에 List<Integer> 활용
  -> List의 자료형이 Integer인 이유 : cartNum의 자료형은 int인데 리액트에서 받아오는 cartNumList는 배열이라 맞추기 위해서
 4. Service 
 - 작성하던대로 똑같이 작성
 5. Controller
  - get / delete는 url parameter로 데이터를 보낼 수 있지만 두번째 매개변수를 활용하여 데이터 전달이 가능
  -> @RequestParam 어노테이션 활용하면 됨!
  -> 사용방법은 @PathVariable이랑 똑같음!
   ex) @RequestParam("cartNumList") List<Integer> cartNumList
  -> 밑의 try, catch문은 똑같이 작성

*리액트*
 6. 체크박스 선택삭제 api 작성
  - url에 스프링의 RequestParam에서 받을 데이터를 두 번째 매개변수 자리에 객체형태로 넣어주기
    -> params 활용! : params는 객체 형태를 띔!
     ex) {params : {'cartNumList' : cartNumList}} 
	=> params는 전체 객체의 key, {'cartNumList' : cartNumList} 는 전체 객체의 value
	=> {'cartNumList' : cartNumList}에서 'cartNumList' 는 key, cartNumList는 value임!

    -> 체크박스 선택 삭제 함수의 매개변수에는 value에 들어가는 값을 넣어줌. => cartNumList 대입
 
 7. 장바구니 선택삭제 함수생성
  - 삭제 시 확인 질문 띄우는 confirm( ) 함수 활용
     -> confirm은 확인 : true, 취소 : false를 반환
  - 변수 하나 생성해서 confirm( ) 저장
  - if문에 조건을 변수의 결과값과 반대를 넣어서 그 결과가 false면 return해서 밑의 코드 진행 멈출 수 있도록 함.
  - if문 하나 더 생성해서 체크박스에서 모든 상품이 해제되어 있으면 빈 배열이기에 빈 배열을 비교하는 조건을 작성
    ex) cartNumList.length === 0
	=> cartNumList가 배열이기에 배열의 길이를 비교하면 배열이 빈 배열인지, 데이터가 들어가있는지 확인 가능!
  - return해서 조건이 false면 return해서 밑의 코드 진행 멈추기
  - if문 밖에서 장바구니 선택 삭제 api 요청 함수 호출 => 매개변수로 cartNumList 전달
  - 선택삭제하고 나서 화면도 다시 리렌더링해서 다시 출력할 수 있게 장바구니 목록 조회 다시 하기
