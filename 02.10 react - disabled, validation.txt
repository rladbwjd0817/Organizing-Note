3. Validation
 - 사용자가 폼에 입력한 데이터가 정의된 규칙을 충족하는지 서버 전송 전에 검증하는 과정
 - 쓰는 이유 : 입력오류를 바로 알려 편리함 제공 + 잘못된 데이터가 서버로 전달 되는 것을 방지하여 보안 및 데이터 정합성 확보
 - 주요 검증 항목 : 필수입력, 형식, 길이(글자수 제한)
* 데이터 유효성검사 - Validation 처리
 : 입력한 데이터가 형식 또는 요구사항에 맞게 작성 되었는지 검사하는 것!
 => 문법 : 정규식
 
ex) 이메일 - 필수 입력, 이메일형식, 최대글자수(데이터베이스 기준)50자,
